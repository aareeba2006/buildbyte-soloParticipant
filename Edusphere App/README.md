## Project Description

**EduSphere** is an all-in-one student productivity and campus-life dashboard built for the BuildByte hackathon. It brings together everything a student needs to manage academics, growth, and wellbeing into a single, clean interface.

### Idea
Students juggle courses, grades, career planning, community involvement, and personal safety across a dozen disconnected apps. EduSphere consolidates all of it into one dashboard so students can plan, track, and grow without switching tools.

### Key Features
- **Dashboard** — at-a-glance overview of everything important
- **Learning Center** — courses, quizzes, flashcards, and study planning
- **Academic Management** — GPA tracking, attendance, timetable, and exams
- **Career Development Hub** — roadmaps, resume builder, and internship tracking
- **Community Hub** — groups, mentors, and collaboration spaces
- **Community Impact** — report campus issues and volunteer opportunities
- **Achievement Center** — XP, badges, and leaderboard for gamified progress
- **Analytics** — trends across study habits and activity
- **AI Assistant** — a personal study copilot
- **Emergency & Safety** — one-tap emergency contacts and safety tips
- **Smart Profile** — a single place for academic identity, skills, and projects
- **Settings** — theme (dark/light mode), notifications, and privacy controls
- **Authentication** — login/signup flow with Google sign-in option

### Tech Stack
- HTML5
- CSS3 (custom properties / theming, responsive layout)
- Vanilla JavaScript (no frameworks)

### Setup / How to Run
No build tools or dependencies required — it's a static site.

1. Clone the repository:
   ```bash
   git clone https://github.com/aareeba2006/buildbyte-soloParticipant.git
   cd buildbyte-soloParticipant
   ```
2. Open `index.html` directly in your browser, **or** serve it locally:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000`.
